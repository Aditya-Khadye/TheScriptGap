# Script Similarity Research

## Setup

cd into the project
`cd similarity_research/diversity_research`

Clone the public google fonts repository to gather google's available font files
`git clone https://github.com/google/fonts.git`

Utilize uv to easily setup your enviorment.

Run `uv sync` to setup your venv or just use `uv run script_diversity_vit_pipeline.py` to run the script.
